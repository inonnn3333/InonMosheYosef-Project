על הפרויקט:

את דפי הנחיתה בניתי על בסיס דפי הנחיתה ששלחו לנו מהמהכללה.

את דף הפורטפוליו בניתי עם שפת התכנות sass בכדי לייעל את העבודה שלי.

זאת בכדי שהדף יראה מסודר, ויפה.
